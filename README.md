一款基于 Capacitor 构建的 Android 刷题应用，支持多种题型、模拟考试、错题管理等功能，帮助你高效备考。

## 一、APP 能做什么？

万能刷题通是一个本地化的刷题工具，所有数据保存在手机本地，不需要联网。它内置了 559 道"习思想"主题的题目（单选 190 道、多选 160 道、判断 189 道、填空 20 道），你也可以导入自己的题库。

APP 支持以下核心功能：

- **顺序练习**：按题型逐题练习，每组 20 题，可查看每组掌握率
- **随机练习**：随机抽题，支持按题型筛选或全部混合
- **模拟考试**：按默认规则自动组卷，也可以自定义各题型数量和分值
- **错题本**：自动收集答错的题目，连续答对指定次数后自动移除（次数可配置，也可关闭自动移出）
- **收藏夹**：手动收藏重点题目，随时回顾
- **多题库管理**：支持创建多个独立题库，数据互不影响，可导入/导出/分享/合并/重命名
- **搜索功能**：在首页搜索框输入关键词，快速定位题目
- **成就徽章**：练习过程中可解锁各种成就徽章，增加学习动力
- **夜间模式**：一键切换深色界面，适合暗光环境使用，设置自动保存
- **主题颜色**：提供经典红、深海蓝、商务黑三种主题，在设置页外观区域一键切换，与夜间模式自由搭配
- **自定义刷题背景**：上传竖版图片作为刷题页面背景，支持调节透明度
- **编辑答案/解析**：练习中发现题目有误，可进入全屏编辑页面查看完整题目信息并修改正确答案和解析，保存后修改即时生效，当前题目重置为未作答状态

## 二、题目类型说明

APP 支持 5 种题型：

| 题型 | 说明 | 答题方式 |
|------|------|----------|
| **单选题** | 4 个选项选 1 个 | 点击选项即可，选对变绿，选错变红 |
| **多选题** | 4 个选项选多个 | 选择时蓝色高亮，点"确认答案"后：选对的选项变绿（绿色圆圈+绿色底纹），漏选的正确选项同样绿色边框和底纹、绿色圆圈，选错的选项变红（红色圆圈+红色底纹），全选对得满分，选不全得半分，错选 0 分 |
| **判断题** | 判断对错 | 点击"正确"或"错误"按钮 |
| **填空题** | 填写空白处 | 在输入框中输入答案，点击"确认答案"；支持多个空，按空或按题计分 |
| **主观题** | 自由作答 | 在文本框写下答案，提交后显示参考答案，始终判定为正确 |

## 三、页面介绍

### 1. 首页（🏠）

打开 APP 看到的第一个页面。顶部显示当前题库名称（点击可切换到其他题库），下方有三个数据卡片显示"已练习"数量、"正确率"和"连续天数"。

中间是功能入口区域：顺序练习、错题本、模拟考试、随机练习、收藏夹，点击即可进入对应功能。

底部有"成就徽章"展示栏和"题库完成度"进度条。

页面最下方是底部导航栏，可以在"首页"、"分类"、"统计"、"设置"四个标签页之间切换。

### 2. 分类页（📂）

按题型展示所有题目，显示每种题型的总题数和已掌握题数。点击某种题型后可以选择每次练习的题数，然后进入练习。

### 3. 统计页（📊）

查看学习数据，包括总练习数、答对数、错题数，以及每种题型的掌握率进度条。下方还有考试记录列表。

### 4. 设置页（⚙️）

设置页面包含六项功能：

**打乱选项顺序**：开启后，做单选题和多选题时 A/B/C/D 选项会被随机打乱（模拟考试同样生效），增加练习难度。注意：解析不会因为选项顺序被打乱而更改，所以更改答案或解析时记得关闭此功能。

**错题本自动移出**：开启后（默认开启），错题本中的题目连续答对指定次数后会自动移除。次数可在 1~999 之间自由设置（默认为 3）。修改次数时，已有错题的"还需做对"次数会自动封顶到新值（例如原本"还需做对 90 次"的题目，改为 3 后会自动变为"还需做对 3 次"，而"还需做对 2 次"的题目不受影响）。关闭此功能后，错题本中的题目将永久保留，不会自动移出。

**夜间模式**：开启后整个 APP 切换为深色界面，适合夜间或暗光环境使用。背景变暗、文字清晰可读，所有按钮和选项都能正常显示。关闭后恢复原来的浅色界面。设置会自动保存，重新打开 APP 后保持上次的选择。

**主题颜色**：提供三种主题风格——经典红（默认）、深海蓝、商务黑。点击对应颜色卡片即可切换，顶部导航栏、进度条、按钮、选中状态等所有主题色区域都会同步变化。主题与夜间模式可以自由搭配，共 6 种组合。设置自动保存，重新打开 APP 后保持上次的选择。

**自定义刷题背景**：可以上传一张竖版图片作为刷题页面（练习和考试）的背景图。只影响刷题页面，不影响首页、设置页、统计页等。背景图片固定在屏幕上，滚动页面或展开解析时不会移动。支持调节背景透明度（0% 为完全不透明，100% 为完全透明，只改变背景图片，题目文字和按钮不受影响），可以随时重新上传图片或清除恢复默认背景（清除前会弹出确认提示）。注意：上传的图片必须是竖图（高度大于宽度），横图或方图会提示换一张。夜间模式和自定义背景可以同时使用。

**题目UI块的透明度**：可以自定义调节题目卡片、选项（包括选中、正确、错误等所有状态）、答案解析等 UI 块底纹背景的透明度（默认 15%，可在 0%~100% 之间调节），值越大越透明，背景图越容易透出。只影响底纹，文字和边框不受影响。设置会自动保存，重新打开 APP 后保持上次的选择。

### 5. 练习页面

进入练习后，顶部显示标题和进度（如"第 3 题 / 共 20 题"）。右上角有收藏按钮（☆/★）和题目导航按钮（☰），最右侧是"完成"按钮（✓）。

每道题答完后会立即显示对错反馈：答对显示绿色，答错显示红色并标出正确答案。如果有解析，可以点击"查看解析"展开。

未答题时，题目下方有一个"显示答案"链接，点击可以直接查看答案（此时选项区会变灰不可点击），再点可隐藏答案恢复答题。

连续答对时会显示🔥连击提示，每连对 10 题会弹出奖励。

作答完成后，底部导航按钮下方会出现"有问题？编辑答案或解析"链接。点击后进入全屏编辑页面，顶部显示完整的题目信息和所有选项（正确答案用绿色标注），下方可以修改正确答案和解析说明。保存后修改即时生效，当前题目会重置为未作答状态，方便用新答案重新练习。修改会直接更新题库数据（包括内置题库和自定义题库），之后分享题库时也会显示修改后的答案。

### 6. 练习结果页

练习结束后展示正确率、答对/总题数、各题型分别的正确数。下方列出所有错题的详细信息（题目、所有选项、你的答案、正确答案、解析）。单选题和多选题的错题卡片会显示完整的 A/B/C/D 选项内容，方便回顾。

底部有三个按钮："回到试卷"可以回顾每道题，"返回首页"回到主页，"重做试卷"可以用相同的题目重新练习一遍。

### 7. 模拟考试页面

考试界面顶部是固定的红色状态栏（不随页面滚动），显示进度（如"3 / 65"）、计时器、收藏按钮和"交卷"按钮，左侧有返回按钮，右侧有导航按钮（☰）。

点击"☰"按钮可以打开题目导航面板。面板顶部有"全部"和各题型的标签页，选择"全部"可查看所有题目，每个题号圆点右上角会显示彩色题型标识（红色"单"=单选、绿色"多"=多选、橙色"判"=判断、紫色"填"=填空、蓝色"主"=主观），已作答的题号显示为黄色，当前题为黄色实心。选择具体题型标签则只显示该类型的题目。点击任意题号可直接跳转。

考试过程中不会立即判对错，所有题目答完后点击"交卷"统一评分。

### 8. 考试结果页

显示总分、各题型得分明细和评语。下方列出错题详情，单选题和多选题会显示完整的选项列表。

底部有"回到试卷"、"返回首页"、"重做试卷"、"导出试卷"四个按钮。"导出试卷"可以将本次考试的题目分享为 JSON 或 TXT 文件。

### 9. 错题本

自动收集你答错的题目。顶部有题型筛选标签（全部/单选/多选/判断/填空/主观），可以快速筛选。

点击右上角"管理"按钮进入管理模式，可以勾选多道题目进行批量操作：全选、分享（点击后弹出格式选择面板，可选导出为 JSON 或 TXT 格式，再通过微信/QQ 等应用分享）、移除选中。

错题本的机制是：同一道题连续答对指定次数后自动从错题本中移除（默认 3 次，可在设置页"错题本自动移出"中调整次数，也可关闭自动移出功能）。

### 10. 收藏夹

和错题本类似，收藏你手动标记的重点题目。在练习或考试中，点击顶部的☆按钮即可收藏/取消收藏当前题目。

同样支持题型筛选、管理模式、批量分享和移除。

### 11. 题库选择页

点击首页顶部的题库名称进入。列出所有题库，点击即可切换。每个题库的练习数据相互独立。每个题库右侧有重命名按钮（✏️，点击后输入新名称即可）和分享按钮（↗）。

右上角"管理"按钮可以进入管理模式，支持批量选择和删除，以及合并多个题库为一个。

底部有"导入新题库"入口，支持上传 JSON 或 TXT 格式的题目文件。下方还有"导入到当前题库"功能，可以将文件中的题目合并追加到当前选中的题库中（内置题库不可追加），当前题库的已答题数量、错题本、收藏夹等数据均会保留。导入页面还提供了"AI 格式化提示词"，可以复制给 AI 工具帮你把任意学习资料转换成 APP 能导入的格式。

### 12. 自定义考试

在模拟考试页面点击"自定义考试"进入。可以自由设置每种题型是否启用、题目数量、每题分值。填空题还支持"按空计分"或"按题计分"两种模式。

页面底部有实时汇总，显示总分和题目总数，方便调整。设置好后点"开始考试"即可。

### 13. 修改默认考试规则

在模拟考试页面的"默认考试规则"标题旁边有一个"修改"链接，点击可以调整模拟考试的默认配置（各题型数量、分值等）。修改保存后，以后每次点"开始考试"都会使用新配置。

### 14. 教程文档页

在首页底部有"教程文档"入口，点击后显示腾讯文档的链接和二维码，可以查看详细的使用教程。

## 四、常用操作流程

### 顺序练习

首页 → 顺序练习 → 选择题型（单选/多选/判断/填空/主观）→ 选择题组（每 20 题一组）→ 开始逐题答题 → 答完查看结果

### 随机练习

首页 → 随机练习 → 选择模式（全部随机/单选随机/多选随机/判断随机/填空随机/主观随机）→ 选择题数（10/20/30/50）→ 开始答题

### 模拟考试

首页 → 模拟考试 → 查看默认规则（可点"修改"调整）→ 点"开始考试"直接开考，或点"自定义考试"自由配置 → 答完所有题 → 交卷 → 查看成绩

### 导入新题库

首页 → 点击顶部题库名称 → 导入新题库 → 给题库起名字 → 选择 JSON 或 TXT 文件 → 导入

如果不知道文件格式，点"查看 AI 格式化提示词"，复制提示词给 AI 工具（如 ChatGPT、豆包等），让 AI 帮你把资料转成标准格式。

### 分享/导出题目

- **分享整个题库**：题库选择页 → 点击题库右侧的分享按钮（↗）→ 选择 JSON 或 TXT 格式
- **分享错题/收藏**：错题本或收藏夹 → 管理 → 勾选题目 → 分享 → 选择格式 → 通过微信/QQ 等分享
- **分享考试试卷**：考试结果页 → 导出试卷 → 选择格式

## 五、一些细节功能

**显示答案**：练习时，还没答题的情况下，题目下方有"显示答案"链接。点击后直接显示正确答案，选项区变灰不可操作。再点"隐藏答案"可以恢复答题状态。

**收藏题目**：练习和考试页面顶部都有☆按钮，点击收藏当前题目（变为★），再点取消收藏。收藏的题目可以在收藏夹中查看和练习。

**题目导航**：练习页面点击☰按钮展开题目导航面板，显示每道题的答题状态（未答/答对/答错），点击题号可跳转。考试页面点击顶部☰按钮打开导航面板，支持"全部"和各题型分标签查看，"全部"视图下每个题号带有彩色题型标识，已答题显示为黄色。

**重做试卷**：练习结果页和考试结果页都有"重做试卷"按钮，点击后会用完全相同的题目重新开始一次练习或考试。

**打乱选项**：在设置中开启"打乱选项顺序"后，单选题和多选题的 A/B/C/D 选项顺序会被随机打乱，每次进入同一道题的顺序不同，更贴近真实考试。

**连击奖励**：练习中连续答对会显示🔥连击数，每连对 10 题弹出奖励提示。

**成就徽章**：首页底部展示 10 种成就徽章，包括初次练习、练习 10/50/100 题、全部完成、连续 3/7 天、正确率 90%、首次模考、模考 35+ 等。

**搜索题目**：首页顶部搜索框输入关键词，可以模糊搜索所有题目内容，点击搜索结果直接进入该题的练习。

## 六、技术信息

### 环境要求

- Node.js 18+
- JDK 21 (如 Eclipse Adoptium JDK 21)
- Android SDK (compileSdk 36, targetSdk 34, minSdk 24)
- Gradle 8.14.3 (项目自带 gradle wrapper)

### 构建步骤

```bash
# 1. 安装依赖
npm install

# 2. 同步 Capacitor 到 Android 项目
npx cap sync android

# 3. 构建 APK
cd android
./gradlew assembleDebug
```

构建完成后，APK 文件位于 `android/app/build/outputs/apk/debug/app-debug.apk`，复制到 `outputs/万能刷题通.apk` 即可分发。

### 技术栈

- Capacitor 8.x — 跨平台移动框架
- Android Gradle Plugin 8.13
- 纯 Web 前端 (HTML/CSS/JS) 单文件 SPA，运行在 WebView 中

### 应用信息

- 包名：`com.quiz.wanneng`
- 应用名：万能刷题通
- 当前版本：`1.0.28`
- 最低 Android 版本：7.0 (API 24)

### 预构建 APK

预构建的 APK 文件位于 `outputs/万能刷题通.apk`。
