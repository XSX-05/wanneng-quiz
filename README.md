# 万能刷题通

> 一款装在手机里的离线题库学习系统 —— 不只是刷题工具，更是你的随身"刷题老师"。

万能刷题通是一款基于 **Capacitor** 构建的 Android 离线题库学习 APP。它围绕"选题库 → 练习或考试 → 自动记录对错 → 复习错题和收藏 → 查看统计 → 按需要调整设置"这条主线，把题库导入、顺序练习、随机练习、模拟考试、错题本、收藏夹、学习统计、答案解析编辑、夜间模式、自定义刷题背景等能力集中到一个本地可离线运行的 HTML 应用中。

- 核心代码集中在单个 HTML 文件 `www/index.html`。
- Android 部分 (`android/`) 主要负责把这个网页包装成手机 APP。
- 数据保存在手机本地，离线可用，导入/导出/分享题库非常方便。

## 主要功能

- **题库管理**：支持内置题库和用户自定义题库，可切换、删除、合并、分享题库。
- **题库导入**：支持导入 JSON 或 TXT 格式题库，自带格式说明与示例，适合把任意资料整理成可刷题内容。
- **练习功能**：支持顺序练习、随机练习、按题型练习、错题练习、收藏题练习。
- **模拟考试**：支持默认模拟考试，也支持自定义考试的题型、题数、分值和计分方式。
- **错题本**：答错的题会自动进入错题本；同一道题连续做对后会自动移出，专攻薄弱题。
- **收藏夹**：可以收藏重要题目、易错题、老师强调的题，并单独练习收藏题。
- **搜索题目**：首页支持按关键词搜索题目，快速定位相关内容。
- **学习统计**：能看到练习数量、正确率、连续学习天数、各题型掌握情况和考试记录。
- **成就徽章**：完成一定学习目标后会自动解锁成就徽章。
- **导入导出与分享**：可导入 JSON/TXT 题库，也可导出或分享题库、错题、收藏题、试卷。
- **AI 提示词帮助**：内置题库格式化 AI 提示词（仅格式化 / 自动生成解析两种模式），方便把普通题库交给 AI 整理为可导入格式。
- **教程入口**：APP 内提供教程文档入口和二维码，方便查看使用说明和最新版安装包。
- **夜间模式**：在设置里可打开深色界面，整 APP 切换为适合夜间使用的风格。
- **刷题背景自定义**：可以只给刷题页面设置自定义竖版背景图，背景固定不随题目滚动。
- **题目答案/解析编辑**：刷题后可以修改题目的正确答案和解析，特别适合修正自制题库中的错误或补充说明。
- **主观题支持**：不仅支持单选、多选、判断、填空，也支持主观题（简答/论述题），可对照参考答案自评。

## 支持题型

APP 当前支持 5 类题型：

| 题型 | 说明 |
| --- | --- |
| 单选题 | 只能选择一个答案。 |
| 多选题 | 可以选择多个答案；自定义考试中可设置"选不全是否给部分分"。 |
| 判断题 | 点击"正确"或"错误"。 |
| 填空题 | 输入一个或多个空的答案；自定义考试中可设置按空计分还是按题计分。 |
| 主观题 | 输入自己的回答，对照参考答案查看；适合简答、论述等题型。 |

> 说明：主观题能力已经内置支持；具体题库中是否包含主观题，取决于当前导入或内置题库的内容。

## 使用流程

### 1. 选择或导入题库

- 首次使用可以直接使用内置题库。
- 也可以在导入题库页导入自己的 JSON / TXT 格式题库。
- 导入多个题库后，可在"题库选择页"切换当前题库，切换后首页、练习、考试、错题和统计都会围绕当前题库进行。
- 不会整理题库格式？可进入"导入帮助页"复制 AI 提示词，把原始资料交给 AI 转成 APP 支持的格式。

### 2. 开始练习

练习支持多种方式：

- **顺序练习**：先选择题型 → 再选每组题量 → APP 自动分好组 → 选择某一组 → 开始刷题。适合"今天刷第 1 组，明天刷第 2 组"这种系统化学习方式。
- **随机练习**：选择随机范围（全部题 / 某种题型 / 错题 / 收藏题）和题量，由 APP 自动抽题。适合快速抽查。
- **按题型练习**：单选、多选、判断、填空、主观题都可以独立进入练习。
- **错题练习**：专门复习错题本中的题。
- **收藏题练习**：专门复习收藏夹中的重点题。

答题时：

- 单选题：点击一个选项即判定。
- 多选题：选择多个选项后确认答案。
- 判断题：点击"正确"或"错误"。
- 填空题：输入每空答案后提交。
- 主观题：输入自己的回答，对照参考答案。

答完后会立即显示对错和解析：

- 答错的题自动进入错题本。
- 收藏的题自动进入收藏夹。
- 刷题页面支持查看解析、上一题/下一题。
- 刷题页面底部"有问题？编辑答案或解析"可即时修改当前题的正确答案或解析。

### 3. 模拟考试

考试和普通练习不一样：普通练习是答完一题立即显示对错，而考试是全部答完统一交卷。

考试页面支持：

- 切题、查看答题卡。
- 计时、收藏题目。
- 未答完会提示是否确认交卷。
- 交卷后显示考试分数、正确率、各题型结果和错题详情。
- 考试错题也会进入错题本。
- 考试结果页可分享/导出试卷。

#### 自定义考试

可以自由设置：

- 哪些题型参加考试。
- 每种题型出多少题。
- 每种题型每题多少分。
- 多选题选不全是否给部分分。
- 填空题按整题计分还是按空计分。

设置完成后 APP 会自动组卷进入考试。

### 4. 复习与统计

- **错题本**：自动收集练习和考试中的错题，可按题型筛选；同一道题连续做对 3 次后会自动移出。
- **收藏夹**：可按题型筛选，可单独练习收藏题。
- **统计页**：展示练习总量、答对数、错题数、正确率、连续学习天数、各题型掌握率、考试记录。
- **成就徽章**：根据练习数量、连续学习天数、正确率、考试表现等自动解锁。

## 个性化能力

### 夜间模式

- 在设置页打开夜间模式后，整个 APP 切换为深色风格：背景、卡片、按钮、输入框等都会变成适合晚上看的样式。
- 关闭后恢复白天模式。
- APP 会记住用户的选择，下次打开仍保持上次选择。

### 自定义刷题背景

- 可选择默认背景，也可上传自己的竖版图片。
- 上传时要求竖图，更适合手机屏幕；横图或方图会提示不合适。
- 上传成功后只在刷题页面显示，不影响首页、统计页、设置页。
- 背景图固定在屏幕后方，题目内容上下滚动时背景不跟随滚动。
- 可调节背景透明度，让背景更"弱"或更"强"。
- 若文字不清，可调节题目卡片和选项的底纹透明度。
- 也可一键清除背景，恢复默认样子。

### 答案解析编辑

- 刷题过程中，如发现答案错误或解析不完整，可点"有问题？编辑答案或解析"进入编辑弹层。
- 编辑弹层会显示完整题干、选项、答案、解析，可直接修改。
- 保存后，该题会按照新的答案和解析显示。
- 对自制题库特别有用：自制题库经常有答案或解析错误，使用此功能可直接在 APP 内修正。

### 选项打乱

- 在设置页可开启"打乱选项顺序"，适用于多选题和单选题练习，避免用户死记答案位置。

## AI 提示词帮助

- 在"导入帮助页"提供两组提示词，分别针对 JSON 格式和 TXT 格式。
- 每组提示词又分为两种模式：
  - **仅格式化题库**：AI 只负责把原始资料整理为 APP 可导入格式，原始资料中有解析就提取，没有就不生成。
  - **自动生成解析**：AI 在格式化题库的同时为每道题自动生成解析（解析质量取决于 AI 模型能力）。
- 把提示词复制给 AI（如豆包、Trae、ChatGPT、Claude、DeepSeek 等），再附上原始资料，AI 会直接输出可导入 APP 的格式化文件。
- 仓库中也保留了一份完整的提示词文件：`outputs/万能刷题通-AI提示词.txt`。

## 数据保存说明

APP 数据主要保存在手机本地（WebView LocalStorage）：

- 自定义题库和当前题库选择。
- 错题本、收藏夹、练习记录和考试记录。
- 成就徽章。
- 夜间模式设置。
- 自定义刷题背景图片。
- 是否打乱选项等个性化设置。

- 优点：完全离线可用，无网络也可使用。
- 缺点：如果卸载 APP 或清除应用数据，本地保存的题库、记录和设置可能会丢失。**建议重要题库及时导出备份**。

## 项目结构

当前主工程目录为 `万能刷题通`，核心功能集中在单文件 Web 应用：

```text
www/index.html       # 核心 HTML/CSS/JavaScript
www/tutorial-qr.jpg  # 教程二维码资源
```

Android 工程负责把 Web 应用包装成手机 APP：

```text
android/
```

仓库中与 APP 相关的关键文件包括：

- `www/index.html`：核心 HTML/CSS/JavaScript 代码。
- `www/tutorial-qr.jpg`：教程二维码资源。
- `outputs/万能刷题通.apk`：预构建 APK（可直接安装）。
- `outputs/万能刷题通-AI提示词.txt`：题库格式化 AI 提示词。
- `android/`：Capacitor Android 工程。
- `package.json` / `package-lock.json`：Node/Capacitor 依赖配置。
- `capacitor.config.json`：Capacitor 应用配置。

## 构建方法

### 环境要求

- Node.js 18+
- JDK 21（如 Eclipse Adoptium JDK 21）
- Android SDK（compileSdk 36，targetSdk 34，minSdk 24）
- Gradle 8.14.3（项目自带 Gradle Wrapper）

### 构建步骤

```bash
npm install
npx cap sync android
cd android
./gradlew assembleDebug
```

构建完成后，APK 文件位于：

```text
android/app/build/outputs/apk/debug/app-debug.apk
```

## 技术栈

- **Capacitor 8.x**：跨平台框架，将 Web 应用包装为 Android 应用。
- **Android Gradle Plugin 8.13**：Android 项目构建。
- **纯 Web 前端（HTML/CSS/JavaScript）**：单文件 `index.html`，内嵌在 Android WebView 中。
- **本地存储为主**：支持完全离线使用；通过系统分享能力支持题库导出与分享。
