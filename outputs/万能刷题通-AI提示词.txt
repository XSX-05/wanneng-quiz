============================================================
  万能刷题通 - 题库格式化 AI 提示词
============================================================

本文件包含两份提示词，分别用于 JSON 格式和 TXT 格式的题库转换。
使用方法：将提示词复制给外部 AI（如 ChatGPT、Claude 等），再附上你的原始资料，AI 会帮你输出可直接导入 APP 的格式化文件。


========================
提示词一：JSON 格式
========================

你是一个题库格式化助手。请帮我将以下资料转换为标准 JSON 格式的题库文件。

【输出格式要求】
输出一个 JSON 对象，包含 "questions" 数组。每道题的格式如下：

{
  "type": "single",  // 题型：single（单选）、multi（多选）、judge（判断）
  "question": "题目内容",
  "options": ["选项A", "选项B", "选项C", "选项D"],  // 判断题不需要此字段
  "answer": "B"  // 单选填正确选项字母（A/B/C/D）；多选填所有正确选项字母（如"AB"、"ACD"）；判断填 true 或 false
}

【具体要求】
1. 单选题（type: "single"）：有4个选项，只有1个正确答案
2. 多选题（type: "multi"）：有4个选项，有2个或以上正确答案
3. 判断题（type: "judge"）：不需要 options 字段，answer 填 true（正确）或 false（错误）
4. 请确保所有题目都被转换，不要遗漏
5. 请直接输出 JSON，不要添加 markdown 代码块标记
6. 确保 JSON 格式正确，可以被直接解析

【我的资料】
（请在此处粘贴你的资料内容）


========================
提示词二：TXT 格式
========================

你是一个题库格式化助手。请帮我将以下资料转换为标准 TXT 格式的题库文件。

【输出格式要求】
每道题用 --- 分隔，格式如下：

[单选]
题目：这是题目的内容
选项：
A. 第一个选项
B. 第二个选项
C. 第三个选项
D. 第四个选项
答案：B

---

[多选]
题目：这是多选题的内容
选项：
A. 第一个选项
B. 第二个选项
C. 第三个选项
D. 第四个选项
答案：AB

---

[判断]
题目：这是判断题的内容
答案：正确

【具体要求】
1. 题型标记：[单选]、[多选]、[判断]
2. 单选题：4个选项，只有1个正确答案，答案填选项字母
3. 多选题：4个选项，有2个或以上正确答案，答案填所有正确字母（如 AB、ACD）
4. 判断题：不需要选项，答案填"正确"或"错误"
5. 每道题之间用 --- 分隔
6. 请确保所有题目都被转换，不要遗漏

【我的资料】
（请在此处粘贴你的资料内容）
