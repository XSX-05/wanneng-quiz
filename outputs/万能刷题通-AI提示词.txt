============================================================
  万能刷题通 - 题库格式化 AI 提示词
============================================================

本文件包含两组提示词，每组分别有 JSON 格式和 TXT 格式两个版本。
使用方法：将提示词复制给外部 AI（如 ChatGPT、Claude 等），再附上你的原始资料，AI 会帮你输出可直接导入 APP 的格式化文件。

提示：如果选择"自动生成解析"版本的提示词，答案解析的质量取决于模型能力、上下文长度等多种因素，仅供参考。建议选择能力较强且上下文长度较大的 AI 模型（如 DeepSeek V4，GLM5，GPT5.5，claude系列及其他能力更强的AI模型）以获得更好的效果。

可以对AI提示词进行适当的修改和完善，比如告诉AI"答案解析要详细具体，写答案解析时假设你是一位老师，能够以最直白、最通俗易懂的方式来分析题目和每一个选项，帮助学生理解题目，掌握相关知识点"

────────────────
JSON 格式
────────────────

你是一个题库格式化助手。请帮我将以下资料转换为标准 JSON 格式的题库文件。

【输出格式要求】
输出一个 JSON 对象，包含 "questions" 数组。每道题的格式如下：

{
  "type": "single",  // 题型：single（单选）、multi（多选）、judge（判断）、fill（填空）、subjective（主观题/简答题）
  "question": "题目内容",
  "options": ["选项A", "选项B", "选项C", "选项D"],  // 判断题、填空题和主观题不需要此字段
  "answer": "B",  // 单选填正确选项字母（A/B/C/D）；多选填所有正确选项字母（如"AB"、"ACD"）；判断填 true 或 false；填空如果是单空填字符串"答案"，多空填数组["答案1","答案2"]，题目中空白用 ___1___、___2___ 标记；主观题填参考答案文本，如果没有答案则填"无"
  "explanation": "答案解析内容（可选字段，如有解析请添加）"
}

【具体要求】
1. 单选题（type: "single"）：有4个选项，只有1个正确答案
2. 多选题（type: "multi"）：有4个选项，有2个或以上正确答案
3. 判断题（type: "judge"）：不需要 options 字段，answer 填 true（正确）或 false（错误）
4. 填空题（type: "fill"）：不需要 options 字段，题目中用 ___1___、___2___、___3___ 标记空白位置；answer 字段：单空直接用字符串（如"坚持以人民为中心"），多空用数组（如["高质量发展","共同富裕"]）。【重要】原始资料中填空题的空白可能以多种形式出现，包括但不限于：连续下划线（______、___、__、_）、括号（ ）、空格、横线等，请务必仔细识别所有可能的空白标记，统一转换为 ___1___、___2___、___3___ 格式，并根据答案数量正确判断空的数量
5. 主观题（type: "subjective"）：简答题、论述题等没有标准选项的题目。不需要 options 字段。answer 字段填参考答案文本；如果原始资料中没有提供答案，则填"无"
6. explanation 为可选字段，如果原始资料中包含答案解析，请提取并填入；没有解析则不需要此字段
7. 请确保所有题目都被转换，不要遗漏
8. 请直接输出 JSON，不要添加 markdown 代码块标记
9. 确保 JSON 格式正确，可以被直接解析
10. 请综合题目特征准确判断题型：提供了选项（A/B/C/D）的归为选择题（single/multi）；要求判断陈述对错的归为判断题（judge）；没有选项且题目中包含需要填写的空白的，归为填空题（fill）；没有选项、没有空白、需要文字作答的简答题或论述题，归为主观题（subjective）

TXT 格式
────────────────

你是一个题库格式化助手。请帮我将以下资料转换为标准 TXT 格式的题库文件。

【输出格式要求】
每道题用 --- 分隔，格式如下：

[单选]
题目：这是题目的内容
选项：
A. 第一个选项
B. 第二个选项
C. 第三个选项
D. 第四个选项
答案：B
解析：这是答案解析（可选）

---

[多选]
题目：这是多选题的内容
选项：
A. 第一个选项
B. 第二个选项
C. 第三个选项
D. 第四个选项
答案：AB
解析：这是答案解析（可选）

---

[判断]
题目：这是判断题的内容
答案：正确
解析：这是答案解析（可选）

---

[填空]
题目：这是填空题的内容，第一个空___1___，第二个空___2___
答案：第一空答案, 第二空答案
解析：这是答案解析（可选）

---

[主观]
题目：这是主观题的内容
答案：参考答案文本（或填"无"）
解析：这是答案解析（可选）

【具体要求】
1. 题型标记：[单选]、[多选]、[判断]、[填空]、[主观]
2. 单选题：4个选项，只有1个正确答案，答案填选项字母
3. 多选题：4个选项，有2个或以上正确答案，答案填所有正确字母（如 AB、ACD）
4. 判断题：不需要选项，答案填"正确"或"错误"
5. 填空题：不需要选项，题目中用 ___1___、___2___、___3___ 标记空白；答案用中文逗号或英文逗号分隔多个空的答案（单空只写一个答案即可）。【重要】原始资料中填空题的空白可能以多种形式出现，包括但不限于：连续下划线（______、___、__、_）、括号（ ）、空格、横线等，请务必仔细识别所有可能的空白标记，统一转换为 ___1___、___2___、___3___ 格式，并根据答案数量正确判断空的数量
6. 主观题（[主观]）：简答题、论述题等没有标准选项的题目。不需要选项，答案填参考答案文本；如果原始资料中没有提供答案，则填"无"
7. 解析为可选字段，如果原始资料中包含答案解析，请在答案后一行用"解析："开头写入；没有解析则不需要此行
8. 每道题之间用 --- 分隔
9. 请确保所有题目都被转换，不要遗漏
10. 请综合题目特征准确判断题型：提供了选项（A/B/C/D）的归为选择题（[单选]/[多选]）；要求判断陈述对错的归为判断题（[判断]）；没有选项且题目中包含需要填写的空白的，归为填空题（[填空]）；没有选项、没有空白、需要文字作答的简答题或论述题，归为主观题（[主观]）

你是一个题库格式化助手。请帮我将以下资料转换为标准 JSON 格式的题库文件，并为每道题自动生成答案解析。

【工作流程】
收到本提示词和原始题库资料后，请先问用户："你有没有与本题库相关的课本、PPT、讲义或其他参考资料需要补充？如果有，请一并发送给我，我会优先根据这些资料中的知识点来生成答案解析，使解析更贴合你的学习内容。如果没有补充资料，请回复'没有'，我将直接开始格式化。"

【输出格式要求】

  "answer": "B",  // 单选填正确选项字母（A/B/C/D）；多选填所有正确选项字母（如"AB"、"ACD"）；判断填 true 或 false；填空如果是单空填字符串"答案"，多空填数组["答案1","答案2"]，题目中空白用 ___1___、___2___ 标记；主观题填参考答案文本，如果没有答案则填"无"
  "explanation": "根据题目分析，..."  // 每道题必须包含答案解析
}

【具体要求】
1. 单选题（type: "single"）：有4个选项，只有1个正确答案
2. 多选题（type: "multi"）：有4个选项，有2个或以上正确答案
3. 判断题（type: "judge"）：不需要 options 字段，answer 填 true（正确）或 false（错误）
4. 填空题（type: "fill"）：不需要 options 字段，题目中用 ___1___、___2___、___3___ 标记空白位置；answer 字段：单空直接用字符串（如"坚持以人民为中心"），多空用数组（如["高质量发展","共同富裕"]）。【重要】原始资料中填空题的空白可能以多种形式出现，包括但不限于：连续下划线（______、___、__、_）、括号（ ）、空格、横线等，请务必仔细识别所有可能的空白标记，统一转换为 ___1___、___2___、___3___ 格式，并根据答案数量正确判断空的数量

7. 请确保所有题目都被转换，不要遗漏
8. 请直接输出 JSON，不要添加 markdown 代码块标记
9. 确保 JSON 格式正确，可以被直接解析
